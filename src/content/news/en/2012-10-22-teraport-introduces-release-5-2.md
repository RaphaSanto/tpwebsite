Last week Teraport introduced version 5.2. of its flexible software toolbox DMU-Toolkit. The lastest release offers the companies customers a broad range of new functions and enhancements. It starts with the new Pro.PathFinder and Pro.PathFreezer integrations into DMU.View Teraport's own 3D visualization viewport. The modules well known core functionality of mounting and demounting simulation and the calculation of swept volumes can be accessed way more comfortable through the new integration workbenches. Especially for Pro.PathFinder the integration comes with a new set of functions like a favourite planning direction and the easy definition of flexible part in a calculations assembly environment.
Marcus Trommen, head of software development is enthusiastic about the integrations: "Mounting and demounting simulation is - from our past experiences - a very complex topic for our users. Giving them an easier access to the process via our workbenches in DMU.View minimizes missunderstandings."
Another innovation is the availability of import options for multiple productstructure files like PLMXML or CATProduct. These allow the processing of structure and/or included geometry for different calculations.
Other tools of the DMU-Toolkit have been optimized regarding performance. Analysis with DMU.Fill, DMU.Space and DMU.ShellSearch operate 2-12 times faster. In addition an optimization of memory usage has been conducted. This enables users that have already profited from the migration to 64bit systems to make another giant leap.
For users, that want to process results of DMU-Toolkit calculations in CAE software another interesting option is available. Data can be written into Nastran CHEXA elements. This is especially interesting for users that use DMU.ShellSearch to create Models of car interiors for acoustics simulation.
