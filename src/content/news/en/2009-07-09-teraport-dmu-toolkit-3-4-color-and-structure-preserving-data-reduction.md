The current release 3.4 of the Teraport DMU-Toolkit contains a number of further enhancements, as, for example, the color and structure-preserving data reduction feature or the re-design of the automated geometric analysis process. As is our custom, these new features have been developed in close collaboration with our clients and are available with immediate effect. Hermann Gaigl, Managing Director of Teraport GmbH, sums up the new features of this current release as follows: “This is yet another important milestone in the ongoing establishment of our modular software architecture in digital prototype construction”.
**Color and structure-preserving data reduction**
With the Teraport DMU-Toolkit data reduction tool it is possible to reduce the CAD data of an engine which, for example, consists of 6 million triangles to less than 1 million triangles within a period of only 10 minutes. Hence, CAD models can be visualized quickly and efficiently. Even though the data volume has been reduced by 90%, there are no visual differences to the original model. This new feature for color and structure-preserving data reduction also offers the user a number of new possibilities for the subsequent utilization of the reduced model. With the retained substructures, it is, for example, possible to accurately identify which component of an engine is colliding with its surrounding environment. Without this extended feature it would only be possible to determine that a collision situation with the surrounding environment exists, but one would not be able to accurately localize which element/component of the engine is responsible for this conflict.
**Re-design of the automated geometric analysis process**
As a result of this re-design, the geometric analysis process can be depicted in an easier and more flexible manner. This means that the amount of time and work required for the integration of the Teraport solution are considerably reduced. The methodical revision of the calculation rules facilitates the simple and direct utilization of the metadata from the PDM system, without the separate and segmented definition of the calculation jobs. In addition to the modules’ 64-bit capability, the SMP (symmetric multi-processor) capability has also been improved. A list of all the new features of the Teraport DMU-Toolkit release 3.4 can be found at download area at the category "list of functions".
